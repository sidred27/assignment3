Anugu Gallery is the fastest, easiest to use WordPress image gallery plugin. Lightbox with Drag & Drop builder that helps you create beautiful galleries with just a few clicks.

== Description ==

We believe that you shouldn't have to hire a developer to create a WordPress gallery. That's why we built a drag & drop gallery plugin that's both EASY, FAST and POWERFUL.

Anugu Gallery is optimized for web and server performance as well as SEO and conversion, because we understand the importance of speed when it comes to image and gallery load times. Especially if your looking to sell from your site.

Works great with the top page builders including: Elementor, Beaver Builder, Divi and more.

 Anugu Gallery is the lite version plugin that comes with all the features you will ever need.

= Drag & Drop Gallery Builder =

You can also add gallery blocks built for the Gutenberg block editor.

The trick to making difficult things easy is to keep the easy things still easy. If you've struggled with WordPress gallery plugins that take video tutorials to learn, you're going to love Anugu Gallery.

= Mobile Ready, SEO Friendly and Optimized for Speed =

Anugu is 100% responsive and mobile-friendly by default. We optimized every query on the front-end and the back-end to ensure maximum speed.

= Easy to Customize and Extend =

You can easily customize with our built-in layouts or create your own.

Most WordPress gallery options are either too basic or require a PhD to figure out. For users, I need something that is easy to use yet powerful. Anugu Gallery is the answer. Its straight forward approach means it is a breeze for users to quickly create galleries.

When it comes to gallery plugins Anugu Gallery has no equal. Solid enough to do the job right, while flexible enough to handle any situation you can throw at it.

== Installation ==

1. Install Anugu Gallery via uploading the files to your server.
2. Activate Anugu Gallery .
3. Navigate to the Anugu tab at the bottom of your admin menu and click the "Add New" button to begin creating, or you can create directly inside the post/page/custom post type of your choice.
